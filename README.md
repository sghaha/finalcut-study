## 컴파운드 클립만 모아 보는법
* 새로운 스마트 모음 만들고 더블클릭한다음에
* 유형 - 다음과 같음 - 복합

## 컴파운드 클립과 일반 클립 사이에 트랜지션을 넣고싶다.
* 컴파운드 클립을 수정해야한다. 
* 컴파운드 클립 마지막부분에 정지프레임을 넣고 트랜지션을 넣자




## Generator(생성기)
* 타이틀아이콘 누르면 같이 나옴
* 제네레이터의 까만부분은 까만게 아니라 빈공간임
* 인스펙터에서 제네레이터에 대한 탭이 생김, 여기서 여러가지 설정하면됨
* 여기다 키프레임 적용하면 여러가지 애니메이션을 만들수 있다





## 흑백 만들기
* 옵션 클릭해서 위로 올려서 복사함
* 컬러탭 가서 색상판 클릭하고 채도 낮춰줌


## 마스크
* 드로우 마스크 설정한다음에 선쪽에 커서 가져다 대고 더블클릭하면 점 하나 더 생김
* 마스크 반전은 인스펙터에 있다



## 카메라 여러대 썼을때
* 카메라 별로 파일명 prefix를 맞춰주는게 좋다. 디렉토리도 나눠주자, 그러면 라이브러리에 보기 좋게 쌓임
* 프록시를 만드는걸 권장한다, 근데 프록시 파일 만드는게 오래걸림 
* 프록시 만들었다면, 설정에서 '멀티캠 클립에 최적화된 미디어 생성'을 체크 해제 해도된다.
* 아직 타임라인에 없는 클립들을 카메라 별로 클릭하고 인스펙터 창에서 앵글 숫자를 먹여준다. ( 이건 꼭 해야한다함)
* 카메라 + 오디오 싱크 맞추는법
  1) 동시에 찍은 클립들을 모조리 클릭하고
  2) 클립 - 클립 동기화
  3) 그럼 클립이 하나로 된다.
  4) 하나로 보이지만 여러파일임
* 여러 캠을 싱크 맞추는법
  1) 모든 파일 선택하고
  2) 파일 - new - 멀티캠 클립
* 클립을 합쳤다면 오디오를 정리한다.
  1) 클립 선택후 인스펙터창 오디오를 클릭한다음에 듀얼 모노로 바꾼다
  2) 필요없는 오디오는 체크 해제하여 비활성화 한다.
* 이제 위에서 만든걸 프로젝트로 가져오자
  1) '보기'를 클릭하면 앵글이라는게 있다
  2) 클릭하면 잘리면서 해당 앵글로 변환된다
  3) 잘리는거 원치 않고 그냥 앵글만 바꾸고 싶으면 옵션 누르고 클릭하자



## 볼륨
* 미리보기 화면 밑에 볼륨 클릭하면 크게보임
* 0은 한순간도 넘어가면안되고
* -6~-12사이에서 노는게 좋고 가끔 -6 넘어가는수준
* 한부분이 소리가 클때
  1) 소리가 큰부분을 R로 지정함
  2) 이때 소리를 낮추면
  3) 경계면도 알아서 조절함
* 옵션 누르고 오디오 선을 클릭하면 키프레임이 찍힌다 이걸로 조절가능 (4개찍고 가운데 내려야한다)
* 이거 말고 바로위에 회색 동그라미 옮기면 된다
* 근데 컨트롤 클릭하면 파형 어떻게 할지도 정할 수 있다
* 이거 말고 오디오에 트랜지션 '크로스 디졸브' 먹여서 할 수 도있다
